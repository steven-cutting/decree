
# 0005: Directory layout: doc/adr

Date: 2025-10-06
Status: Accepted

## Context

Common tools expect ADRs under `doc/adr`.

## Decision

Default ADR directory is `doc/adr`; CLI allows overrides.

## Consequences

* Parity with existing repos.
* Simple defaults.

## Alternatives considered

* `docs/adr`, `adr/`.
